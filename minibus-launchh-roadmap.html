Launch & Monetization Roadmap 100% Free Hosting

Complete guide for deploying and monetizing MiniBus Egypt without spend. Track your progress with the interactive checkboxes below.

Step 1: Code Versioning with GitHub ($0)

Establish git version control and push the project files to a private or public GitHub repository to set up automatic deployment integrations.

How-To Steps:

  1. Open your terminal/command prompt at the root directory of MiniBus Egypt.
  2. Initialize the repository and commit your files:
    git init
    git add .
    git commit -m "initial commit: minibus egypt production v1"
  3. Create a new repository named minibus-egypt on your GitHub Dashboard.
  4. Set up the remote and push the branch:
    git branch -M main
    git remote add origin https://github.com/YOUR_USERNAME/minibus-egypt.git
    git push -u origin main
✓ Instant: Repository established and secure. Ready for automated deploy.

Step 2: Continuous Deployment via Cloudflare Pages ($0)

Deploy the static files globally to a lightning-fast CDN with permanent SSL protection. Cloudflare Pages is completely free with no limits on traffic.

How-To Steps:

  1. Log in or register at the Cloudflare Dashboard.
  2. Navigate to Workers & Pages > Create Application > Pages tab.
  3. Click Connect to Git, select your GitHub account, and select the minibus-egypt repository.
  4. Configure deployment configurations exactly as follows:
    • Framework preset: None (We are pure Vanilla HTML/CSS/JS)
    • Build command: Keep completely blank (empty)
    • Build output directory: / (or leave empty, which defaults to root where index.html sits)
  5. Click Save and Deploy. Cloudflare will build the static site.
✓ SLA: Deployment takes ~2-5 minutes. You will receive a permanent live URL like minibus-egypt.pages.dev. Future git pushes auto-deploy instantly.

Step 3: Create Mandatory Pages for Google AdSense ($0)

Google AdSense automatically rejects sites that lack a standard Privacy Policy and About page. Create them to pass eligibility screenings.

How-To Steps:

  1. Verify that privacy-policy.html and about.html exist at your project's root folder.
  2. Check that the footer of index.html has clear, accessible links pointing to them.
  3. Commit these files and push to GitHub so your Cloudflare Pages live URL has them ready:
    git add .
    git commit -m "add policy and about pages for adsense approval"
    git push
✓ Instant: Cloudflare will deploy the changes automatically in 30 seconds.

Step 4: Register & Apply on Google AdSense ($0)

Initiate the AdSense application process. Google will inspect your content, layout, traffic value, and compliance pages.

How-To Steps:

  1. Navigate to Google AdSense and click Get Started.
  2. Enter your live Cloudflare Pages URL (or custom domain) and your payment region (Egypt).
  3. Copy the AdSense verification code snippet and paste it into the of index.html.
  4. Create an ads.txt file in your root directory containing Google's verified client string to prove ownership.
  5. Submit your application for review.
⌛ SLA: Review takes 2 to 14 days. Minimum payout is $100, paid directly via Bank Wire (supported by Egyptian banks in EGP/USD).
